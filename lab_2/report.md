## Отчет о выполнении лабораторной работы №2

В данной лабораторной работе были созданы и протестированы генераторы случайных последовательностей на языках программирования C++ и Java.

Генераторы были использованы для создания генераторов случайных чисел (ГСЧ). Сгенерированные последовательности были записаны в файл sequence.json.

Во второй части работы были проведены три теста NIST на сгенерированных последовательностях:

1. Частотный побитовый тест.
2. Тест на одинаковые подряд идущие биты.
3. Тест на самую длинную последовательность единиц в блоке.

Для каждой последовательности были получены результаты тестов. Для условия случайности последовательности требуется, чтобы значение P-значения было больше 0.01.

Результаты тестов для последовательности на языке C++:
1. Частотный побитовый тест - 0.8596837951986662.
2. Тест на одинаковые подряд идущие биты - 0.37813571020101566.
3. Тест на самую длинную последовательность единиц в блоке - 0.718013202470707.

Результаты тестов для последовательности на языке Java:
1. Частотный побитовый тест - 0.7236736098317631.
2. Тест на одинаковые подряд идущие биты - 0.21140702817801685.
3. Тест на самую длинную последовательность единиц в блоке - 0.344312071738815.

На основании результатов тестов можно сделать вывод, что обе сгенерированные последовательности являются случайными.
